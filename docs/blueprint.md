# **App Name**: FlashFlow

## Core Features:

- Deck Creation: Create custom flashcard decks for different subjects.
- Card Management: Add and edit individual flashcards within a deck, with front and back content.
- Card Review: Review cards with a flip action to reveal the answer on the back.
- Progress Tracking: Mark cards as known or unknown to track learning progress.

## Style Guidelines:

- Primary color: Soft blue (#89B2D8) for a calming and focused learning environment.
- Background color: Light gray (#E6E6E6) to provide a clean, non-distracting backdrop.
- Accent color: Slightly darker blue (#6A93B8) for interactive elements and progress indicators.
- Body and headline font: 'PT Sans' for a modern and readable text.
- Simple, clean icons for navigation and actions, in the primary blue color.
- Clean, minimalist layout to prioritize content and reduce distractions.
- Smooth, subtle animations for card flipping and transitions.
